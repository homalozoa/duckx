# DouX Weight Tracker

A responsive Vue 3 + Express.js app for tracking pet duck DouX's weight in grams.

## Commands

Install dependencies:

```bash
npm install
```

Run frontend and backend during development:

```bash
npm run dev
```

Run tests:

```bash
npm test
```

Run a single test file:

```bash
npm test -- server/api.test.js
```

Build frontend:

```bash
npm run build
```

Start backend:

```bash
npm start
```

## Data

Records are stored in `data/data.json`. Weight values are stored and displayed in grams.
